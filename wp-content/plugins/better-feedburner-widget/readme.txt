== Description ==

A customizable widget allowing you to place a Feedburner email subscribe form in any widgetized area of your site.

= Current features =

1. Create a Feedburner email subscription widget
2. Customize the widget with any content you like
3. Add subscriber stats with 1 click

= Note =
This requires a <a href="http://feedburner.google.com/">Feedburner</a> account and for email subscriptions to be turned on. In addition, the stats button must also be turned on if using the feature.

== Installation ==

1. Upload the zip file to the `/wp-content/plugins/` directory
2. Unzip
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Configure and customize the widget on the 'Widgets' page.
